# Hello World

## Overview
A styled landing page that introduces the vo1d gamertag for AIML 1870.

## Features
- Centered greeting displaying "Hello from vo1d!"
- Gradient text effect on the gamertag (blue-to-purple)
- Pulsing radial glow background animation
- Fade-in entrance animation
- Dark theme (#0a0a0a background)
- Fully responsive, single-file HTML project

## Tech
- HTML5 / CSS3
- CSS animations (`@keyframes fadeIn`, `@keyframes pulse`)
- CSS `background-clip: text` for gradient text
- No JavaScript, no external dependencies
